Class: "Instance"

Definition

Resource reflecting an individual, material embodiment of a Work.
CHECKME I'm not too thrilled with the introduction of Instance in the model because the definition is too attached to the "library" way of thinking about books: the "individual, material embodiment" is in the case of the TibSchol model really the <MANUSCRIPT> (or a part within it). If we want to record the fact that several manuscripts transmit the same "version" of a work, this would not be on the material but on the intellectual level.
